Embedding Models

Embedding models are running.

Available models: NVEmbed and SciBertEmbed. Simply call them with their respective model name, for example: 'http://10.230.252.6:7777/api/SciBertEmbed'.

Sample Usage


import requests
import json

HOST = "http://localhost:7777"
SUB = "/api/NVEmbed"
URL = HOST + SUB

data = {'input': ['how are you', 'i am good'], 'type': 'documents'}
# data = {'input': 'how are you', 'type': 'query'}

response = requests.post(URL, json=data, verify=False)
response_data = response.json()
print(json.dumps(response_data))
